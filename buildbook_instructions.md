This book is built using [Quarto](https://quarto.org/docs/books/)

Install Quarto:

```{r}
install.packages("quarto")
```

After editing a quarto file, commit the changes and then push to the repo - Github should automatically render the file. 

The book is using a minimum amount of R packages - if you include something that requires an R package installed, let Adele know and I'll update the dependencies so that the book will deploy.

I've numbered the quarto files according to the chapter progress in the book. If you think there should be a new section in between existing files, feel free to add a new file and then rename the rest accordingly.
